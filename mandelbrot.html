Mandelbrots

Description

Ever since I picked up a copy of Roger Penrose's "The Road to Reality" I have had a special interest in the mandelbrot fractal.

So far I have made two mandelbrot viewers:
- One in written in python and can generate high resolution images, given specific coordinates.

- The other implementation is a unity shader that can be explored in realtime. The limitation of the shader is that it can not be explored as deeply (32 bit floating point numbers) as the python program.
